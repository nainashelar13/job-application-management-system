ATS Dashboard

A responsive Job Application Management System built using React.js and Tailwind CSS.

Features :

Login Page
Dashboard
Job Listings
Candidate Listings
Candidate Details Modal
Search Functionality
Responsive UI

Tech Stack :

React.js
Vite
Tailwind CSS
React Router DOM
Recharts

Run Project :

npm install

npm run dev
